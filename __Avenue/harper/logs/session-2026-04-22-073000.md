## Next Steps to Get the Data

**To run the NYT retrieval (recommended):**
1. Register at https://developer.nytimes.com/ (5 minutes, free)
2. Create an app, copy the API key
3. Run: `python3 Paper/scripts/nyt_per_axis_query.py --api-key YOUR_KEY`
4. Wait ~90 minutes (or run overnight)
5. Run: `make data-refresh`

**To use BigQuery instead (GDELT path):**
1. Go to console.cloud.google.com/bigquery (just needs Google account)
2. Run `python3 Paper/scripts/gdelt_per_axis_query.py` to print the SQL
3. Paste into BigQuery console, run, export CSV
4. Save to `Paper/data/raw/gdelt_per_axis_raw.csv`
5. Run: `make data-refresh`
