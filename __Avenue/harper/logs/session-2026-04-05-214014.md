# Session Log - 2026-04-05 21:40:14

## What Was Wrong / What Was Requested

The user asked for a **proper LaTeX bibliography** for each document under `supporting_material/`. Those papers were Pandoc-style **article** sources with a long **numbered “Works cited”** paragraph (mostly URLs), not structured BibTeX.

## How I Fixed It / What I Did

1. **Tooling** (`supporting_material/_tools/`)
   - `works_cited_to_biblatex.py` — locates the `\paragraph{…Works cited…}` block, splits numbered entries (including cases where `N.` was glued on the same line as a prior URL), and writes a sibling **`.bib`** file of `@online` / `@misc` entries with `title`, `url`, and `urldate = {2026-04-05}`.
   - `patch_tex_biblatex.py` — (for future fresh docs) inserts **biblatex** before **bookmark/hyperref** and replaces the works-cited paragraph with `\nocite{*}` + `\printbibliography[title={Works cited}]`.

2. **Per-document setup** (five `.tex` files)
   - **Prohibition Era Violence and Gangs** → `Prohibition_Era_Violence_and_Gangs.bib` (56 entries)
   - **Analyzing Late 20th Century Crime Surge** → `Analyzing_Late_20th_Century_Crime_Surge.bib` (84)
   - **Lead, Highways, and Systemic Harm** → `Lead_Highways_and_Systemic_Harm.bib` (84)
   - **The Architecture of Violence…** → `The_Architecture_of_Violence_A_Multidimensional_F.bib` (83)
   - **`__archived/Lead, Violence, and State Complicity`** → `Lead_Violence_and_State_Complicity.bib` (63)

3. **Preamble**
   - `\usepackage[backend=biber,style=authortitle,maxbibnames=99,doi=false,isbn=false,url=true]{biblatex}`
   - `\addbibresource{<matching>.bib}` **before** `\usepackage{bookmark}` (biblatex requires **hyperref after biblatex**).

4. **End of document**
   - Replaced the inline list with:
     - `\nocite{*}` (include all `.bib` entries without yet adding `\cite{…}` in the body)
     - `\printbibliography[title={Works cited}]`
     - `\label{works-cited}`

5. **Verification**
   - Full **pdflatex → biber → pdflatex** cycle succeeded for **Prohibition Era Violence and Gangs** and **Analyzing Late 20th Century Crime Surge**; **biber** succeeded for **Lead, Highways, and Systemic Harm** (that project still has unrelated **pgfplots** coordinate warnings).

## Challenges Encountered

1. **Package order:** Initial placement of biblatex *after* `bookmark` (hyperref) caused aux/bbl issues; fixed by loading biblatex **before** `bookmark`.
2. **Job names with spaces:** `biber "File Name"` and `pdflatex "File Name.tex"` must use quoted names.
3. **Glued reference numbers:** Parser required a preprocessor to insert newlines before ` N. ` when concatenated with URLs on one line.
4. **Entry count drift:** Analyzing vs Architecture differ by one item (84 vs 83) after split — reflects source text, not tool failure.

## Next Ideas (6 Ideas)

2. Switch to `style=apa` or `ieee` if a journal mandates it.
3. Add `sorting=none` to preserve original numbered order instead of alphabetical **nty**.
